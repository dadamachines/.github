# dadamachines

Open-source music technology from Berlin. We build the [**TBD-16**](https://dadamachines.github.io/ctag-tbd/) — a standalone desktop instrument with 50+ DSP plugins, standard MIDI, browser UI, and Ableton Link. Built on [CTAG TBD](https://github.com/ctag-fh-kiel/ctag-tbd) by Robert Manzke. Fully open source (GPL/LGPL).

🌐 [dadamachines.com](https://dadamachines.com) · 📖 [Docs](https://dadamachines.github.io/ctag-tbd/) · 💬 [Contact](https://dadamachines.com/contact/)

## Firmware & Tools

- [**ctag-tbd**](https://github.com/dadamachines/ctag-tbd) — ESP32-P4 audio DSP engine, UI/UX, and docs
- [**tbd-usb-msc**](https://github.com/dadamachines/tbd-usb-msc) — USB disc mode for firmware & SD card access
- [**dada-tbd-firmware**](https://github.com/dadamachines/dada-tbd-firmware) — Pre-built firmware binaries
- [**dada-tbd-tui**](https://github.com/dadamachines/dada-tbd-tui) — Terminal-based update wizard

## Apps & Templates

- [**dada-tbd-app-template**](https://github.com/dadamachines/dada-tbd-app-template) — Starting point for RP2350 apps (Arduino & PlatformIO)
- [**dada-tbd-doom**](https://github.com/dadamachines/dada-tbd-doom) — DOOM 1 running on the TBD-16

## Get Involved

Write a [DSP plugin](https://dadamachines.github.io/ctag-tbd/plugins/step-by-step.html), build a [controller app](https://dadamachines.github.io/ctag-tbd/apps/getting-started.html), or create your own [hardware product](https://dadamachines.github.io/ctag-tbd/hardware/30_custom_integration.html) on the platform.
